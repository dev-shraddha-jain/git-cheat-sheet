Setup & Configuration

CommandDescription
git --versionCheck Git version

Basic Commands

CommandDescription

Setting Up SSH Keys (Mac)

StepCommand / Description
Generate a new SSH key ssh-keygen -t ed25519 -C "your@email.com"
If ed25519 is not supported, use rsa:
ssh-keygen -t rsa -b 4096 -C "your@email.com"
Start ssh-agent in background eval "$(ssh-agent -s)"
Add key to ssh-agent ssh-add ~/.ssh/id_ed25519
If using RSA, ssh-add ~/.ssh/id_rsa
Copy public key to clipboard pbcopy < ~/.ssh/id_ed25519.pub
Or pbcopy < ~/.ssh/id_rsa.pub
Add key to GitHub/GitLab account Go to your account's SSH Keys section and paste the public key.
Test connection (GitHub) ssh -T git@github.com

For custom configuration, edit ~/.ssh/config
Host github.com
AddKeysToAgent yes
UseKeychain yes
IdentityFile ~/.ssh/id_ed25519
